🚗 Car Sales Dashboard (Excel Project)

📌 Overview
This project is an interactive Car Sales Dashboard built using Microsoft Excel. It focuses on analyzing sales performance, customer trends, and product insights to support data-driven decision-making.

📊 Key Features
Data cleaning and preprocessing (handling missing values, duplicates, and data types)
Interactive dashboard with multiple charts and KPIs
Sales analysis by brand, region, and time
Customer and product segmentation insights
Dynamic filtering using slicers

🛠️ Tools & Techniques
Microsoft Excel
Pivot Tables & Pivot Charts
Data Cleaning Techniques
Dashboard Design & Visualization
Basic Excel formulas for calculations

📈 Insights Generated
Identified top-performing car brands and models
Analyzed sales trends over time
Compared regional sales performance
Highlighted key factors affecting revenue

📂 Project Structure
Car_Sales_Data.xlsx → Raw and cleaned dataset
Dashboard.xlsx → Final interactive dashboard

🚀 How to Use
Open the Excel dashboard file
Use slicers and filters to explore data
Interact with charts to gain insights

🎯 Project Purpose
This project was created as part of my learning journey in Data Analysis, focusing on building strong foundations in Excel dashboards and business insights.
